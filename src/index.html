About

Welcome to a brand new Github Pages template! This project was created to help kick-start other projcts you're working on. You may use this template as is or make changes to it for your own project.

Features

In addition to a beautiful, simple, and responsive template for your new website, this project with a complete build pipeline and development environment out of the box. You can make changes to your project, run development and production builds, and deploy to Github Pages with a few simple NPM scripts.

Libraries

This project uses Node and the Node Package Manager (NPM). The package manager is configured to include Gulp, jQuery, Bootstrap, and SASS. All of these technologies are included using NPM and are incorperated into your Gulp build pipeline. This makes it easy to start building fast without having to spend time setting up the basic project structure.
