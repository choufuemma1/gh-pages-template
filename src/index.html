About

Welcome to your brand new Github Pages template! This general purpose template is free to use and can be adapted as a personal website, web application, or can be used to kick-start a new project.

Features

In addition to a beautiful, simple, and responsive template for your new website, this project comes with a complete build pipeline and development environment. You can make changes to your project, run development and production builds, and deploy to Github Pages in seconds.

Libraries

This project uses Node and Node Package Manager (NPM) to include Gulp, SASS, jQuery, and Bootstrap. All of these technologies are incorperated into your Gulp build pipeline. This makes it easy to start building fast without having to spend time setting up the basic project structure.
